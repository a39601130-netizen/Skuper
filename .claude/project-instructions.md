# Правила работы с проектом Budget Bot

## Процесс разработки и деплоя

При работе над ботом (добавление фич, исправление багов):

1. **Локальное тестирование**
   - Все изменения сначала тестируются локально на компьютере
   - Бот запускается и проверяется работоспособность

2. **Деплой после одобрения**
   - После одобрения пользователем используй команду `/update-bot`
   - Команда автоматически:
     - Коммитит все изменения
     - Пушит на удаленный репозиторий
     - Обновляет код на сервере
     - Перезапускает бота
   - НЕ делай update-bot без явного одобрения пользователя
