Описание: 
Этот файл содержит последовательных команд, для обновления бота на удаленном сервере.

Инструкция
1. Закомитить изменения
2. Пушить изменения
3. Используя для подключения к удаленному серверу команду "ssh bot" в деректории ~/Artur/Skuper получить изменения с помощью команды "git pull" и перезапустить бота внутри tmux
4. Проверить что он запустился успешно
